Cursos -> Tecnologia em Sistemas para Internet

APRESENTAÇÃO

O curso de Tecnologia em Sistemas para Internet é composto por 5 módulos cursados em 2,5 anos. Os módulos do curso tratam das principais áreas do desenvolvimento web, como Modelagem da Interface do Usuário (front-end), Desenvolvimento da Interface do Usuário (front-end), Desenvolvimento de Comércio Eletrônico (back-end), Desenvolvimento de Portais Corporativos (back-end) e Arquitetura e Engenharia de Software (documentação).

No módulo de Modelagem da Interface do Usuário o aluno aprende técnicas e ferramentas para modelar interfaces do usuário e definir os aspectos da arquitetura da informação, a utilizar ferramentas de tratamento de imagem e vídeo (Adobe Photoshop e Premiere) e ferramentas de gerenciamento de conteúdo (WordPress), a definir estratégias e utilizar ferramentas de marketing digital na web.

ÁREAS DE ATUAÇÃO

O Profissional formado no curso de Tecnologia em Sistemas para Internet poderá atuar, dentre outros, como Analista de Sistemas, Programador Java EE, Programador PHP, Programador .NET, Desenvolvedor Java Web, Web Designer, Desenvolvedor Web Fullstack, Desenvolvedor Front-end, Desenvolvedor back-end, Analista de Suporte Web, Gerente de Projetos Web, Arquiteto de Informação para Web, Desenvolvedor Web etc.

MERCADO DE TRABALHO

DIFERENCIAIS

Como a maioria dos sistemas de software hoje convergem para a web, possibilitando o acesso a partir de qualquer dispositivo conectado à Internet, a área de Sistemas para Internet possui grande quantidade de postos de trabalho disponíveis e em franco crescimento. Estima-se que nos próximos anos continuará sendo uma das áreas da Informática mais promissoras na oferta de empregos.

FALE COM O COORDENADOR

O curso aborda as principais áreas necessárias para formar um profissional completo capaz de projetar e criar deste o design, a interface do usuário e a parte do site que executa em servidores de aplicação, integrando recursos de forma segura e otimizada por meio das principais ferramentas e linguagens computacionais. Conta com um grupo de professores atuantes no mercado e engajados em pesquisa e inovação, contribuindo diretamente para manter a Matriz Curricular atualizada e alinhada com o mercado de trabalho. O ambiente acolhedor e os recursos tecnológicos disponíveis entregam ao aluno o ambiente ideal para desenvolver seu potencial técnico, criativo, crítico e ético. O aluno aprenderá a projetar, documentar e desenvolver deste sites simples até sites de comércio eletrônico e portais corporativos que integram aplicações e recursos.
